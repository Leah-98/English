# 四大词性

四大词性与七类成分的关系（关系1）：

- 名词：主语(S)、宾语(O)、表语(P)、补语(C)
- 形容词：定语、表语(P)、补语(C)
- 副词：状语
- 动词：谓语(V)

## 动词

### 分类

任何复杂结构都不能作为动词使用

#### 系动词

系动词：本身不表示任何动作，用于将前后两个结构联系起来，主语+系动词之后的内容为表语。

常用的系动词：be动词、keep、seem、appear、become、feel、get、grow、remain、smell、loook、sound、taste、stay、go

be动词：is、are、am、was、were、been、be、being

系动词的特点1：之后可以接名词和形容词（表语只能为名词或形容词）

系动词的特点2：将系动词替换为be动词，句子的语义基本不变

#### 助动词

不能用作谓语动词，不具有实际意义

#### 情态动词

不能用作谓语动词，不具有实际意义

### 使用

#### 时态

|      | 一般式             | 进行式                     | 完成式                   | 完成进行时                          |
| ---- | ------------------ | -------------------------- | ------------------------ | ----------------------------------- |
| 现在 | 一般现在时 do      | 现在进行时 am/is/are doing | 现在完成时 have/has done | 现在完成进行时 have has been doing  |
| 过去 | 一般过去时 did     | 过去进行时 was/were doing  | 过去完成时 had done      | 过去完成进行时 had been doing       |
| 将来 | 一般将来时 will do | 将来进行时 will be doing   | 将来完成时 will had done | 将来完成进行时 will have been doing |

一般现在时表述客观事实、习惯等，现在进行时表述进行态

现在完成时重视动作的结果，一般过去时重视动作发生的时间

如果要明确表示两个过去动作发生的先后顺序，使用过去完成时表述先发生的动作（过去的过去），使用一般过去时表述后发生的动作（过去），比如When I arrived home, Tom had left

##### 虚拟语气

表示虚拟语气的方法：将句子中的谓语后退一个时态；过去完成时 had done <-- 一般过去时 did <-- 一般现在时 do

无论主语是单数还是复数，在虚拟语气中只能将谓语be动改写为was

在谓语动词前加上情态动词 would/might/should/could 表示对”将来发生动作“的虚拟语气

if...(then)结构的虚拟语气：

- 【现在】if + 一般过去时，then + would/might/should/could + 动词原形
- 【完成】if + 过去完成时，then + would/might/should/could + 现在完成时
- 【将来】if + should/were to + 动词原形，then + would/might/should/could + 动词原形
- 主句和从句也可以表达对不同时间的虚拟，比如：If they had set off yesterday(过去), they would be here now(现在). / If we shouldn't have a date this afternoon(将来), I would go shopping now(现在).

#### 主谓一致

主语为**不可数名词**时，谓语动词只能为单数

**集体名词**是用于表示一类人或者一类事物，比如familiy\team\company\crowd\class\couple

- 当这个集体被作为一个整体看待时，谓语动作应为单数
  - 例如：My family is going to travel abroad.
- 当集体中的成员被作为个体看待时，谓语动词应为复数
  - 例如：My family are taking different planes to Paris.

由于**动名词、动词不定式和名词性从句**这三类名词性结构均没有复数形式，所以当这三类结构用作主语是，谓语动词只能为单数形式

**不定代词**：

- one\either\neither\each 表示单个的人或事物，谓语动词为单数
- every-\no-\some-\any- 复合不定代词，表示单个的人或事物，谓语动词为单数
- many\both\few\others 表示多个人或事物，谓语动词为复数
- some\any\none\all\more\most 表示单数又能表示复数，谓语动词的形式由介词of后的名词或代词决定
  - All of my friends like the movie. All of the money was wasted on the project.
  - 在"X of Y"的结构中，并不会将其中的 of Y 看作介词短语，而会将 X of 看作一个修饰Y的形容词

主语连接：

- 当主语是and连接的两个（或以上）的事物时，谓语动词使用复数形式
- 当主语是or或者nor连接的两个（或以上）的<u>单数事物</u>时，谓语动词使用单数形式
- 当主语是在单数事物或者复数事物之间选择时，谓语动词与更靠近它的那个事物单复数形式一致

### 被动语态

| 时态       | 主动语态        | 被动语态             |
| ---------- | --------------- | -------------------- |
| 一般现在时 | do              | is/am/are done       |
| 一般过去时 | did             | was/were done        |
| 一般将来时 | will do         | will be done         |
| 现在进行时 | is/am/are doing | is/am/are being done |
| 过去进行时 | was/were doing  | was/were being done  |
| 现在完成时 | have/has done   | have/has been done   |
| 过去完成时 | had done        | had been done        |

被动语态中be动词的时态与主动语态中谓语动词的时态应保持一致，而被动语态中过去分词的形态永远是done

#### used的用法

1. 形容词，“习惯的”：I am used to the hot days.

2. 动词，“过去常做的”：He used to go shopping during the weekend.

3. 动词，“使用”：The internet can be used to do what traditional advertising does.

## 副词

副词的修饰对象：

- 可以修饰名词和代词之外的其他结构。
- 当修饰形容词、副词、短语、动词不定式等时，一般置于句中（被修饰词之前）
- 当修饰句子时，一般置于句首
- 当修饰动词时
  - 程度副词、频率副词：置于句中（谓语之前）
    - 例外：句中的谓语是be动词时，频率副词只能置于谓语之后
  - 方式副词、时间副词、地点副词：置于句末
    - 例外：为了让别人知道方式副词在句中修饰的到底是什么结构，有时也会让副词尽量靠近其修饰的结构，比如：The boy placed cherries on the cake carefully --> The boy carefully placed cherries on the cake

副词的分类：

- 评论副词：表示说话者的态度和意见
  - 必须放在句首
  - 必须用逗号将其与句子分开
  - 比如：fortunately\finally
- 方式副词：为动词添加“该动作发生的方式”这个附加信息
  - 即回答how开头的问题
  - 一般用于修饰动词
  - 比如：fast\quietly
- 频率副词：为动词添加“该动作发生的频率”这个附加信息
  - 即回答how often开头的问题
  - 一般用于修饰动词
  - 比如：sometimes\never
- 程度副词：为目标词添加“程度”这个附加信息
  - 即回答how much开头的问题
  - 一般用于修饰形容词、副词
  - 比如：very\quite\rather(相当)
- 时间副词：为动词或句子提供“该动作发生的时间”这个附加信息
  - 即回答when开头的问题
  - 一般用于修饰动词、句子
  - 比如：today\tomorrow
- 地点副词：为动词或句子提供“该动作发生的地点”这个附加信息
  - 即回答where开头的问题
  - 一般用于修饰动词、句子
  - 比如：outside\here
- 连接副词(特殊)：
  - 并不修饰任何结构，作用是连接前后两个句子
  - 不可以删除，会影响语义
  - 通常用逗号将其与后面的句子分离
  - 常用的包括：however\therefore\nevertheless\moreover\thus

### Still Yet Already

三者是时间副词：

- Still：一件事在”现在“还在进行

- Yet：一件事在”将来“某个时间才会发生
  - 注意：通常只用于否定句或疑问句

- Already：一件事在”过去“已经发生了

定语：形容词、介词短语、定语从句等可以充当形容词使用的结构

及物动词之后的结构为介宾(Op)、系动词之后的结构为表语

## 非谓语动词

### 介词短语

介词是用来建立一个名词与其他词（包括名词、动词、形容词等）关系的一类词

介词短语：介词+名词（名词性结构），组成表示时间、状态、目的、原因的介词短语



#### at\on\in 表示地点时的区别

at：一维、表示点

on：二维、表示平面的表面

in：三维、表示物体的内部

当乘坐飞机、火车、公共汽车等大型交通工具时，也可以看作是位于一个较大的平台表面之上，可以使用on：on the plane\on the train\on the bus

当使用介词in时，通常特指在某个事物或区域的内部：in the library

当表示乘坐私家车、小船等小型交通工具时，我们会突出人所处的位置是在这类交通工具的内部，因此也会使用介词in：in the car\ in the boat



#### at\on\in 表示时间时的区别

这三个介词按照 at ->on->in 的顺序，表示的时间长度逐渐增加。

介词at通常用于表示一天中某个特定的时间，即“某个时间点”：at 8:00\ at the moment

介词on通常用于表示“某一天”：on Tuesday\on Valentine's Day

介词in通常用于表示一个更长的时间段，比如周、月、年、世纪等：in 1998\in September



#### 判断介词短语在句子中的成分

假设其时定语、状语、补语中的任意一类，翻译成中文后看看是否符合逻辑

例子：Dad put a parcel under the table

1. under the table是定语：修饰 a parcel，a parcel under the table翻译后为”桌子下面的包裹“，整句为”爸爸放桌子下面的包裹“
2. under the table是状语：状语可以省略，省略后为Dad put a parcel，有语义问题
3. under the table是补语：可以省略主语和谓语，在宾语和补语之间加入be动词后得到a parcel was under the table，翻译整句为”爸爸把包裹放在了桌子下“

根据上面的分析，under the table应该为补语



#### 补语对介宾做补充说明

称介词短语中的名词为介宾，因为介宾是宾语中的一种，补语同样可以对介宾作补充说明。

例子：I looked at Andy sitting there helplessly

上面的例子中，sitting there helplessly为 Andy的介宾补语，可以与Andy组成句子：Andy was sitting there helplessly



#### 可以作为介词的词，其实使用了其他词性的情况

例子：I get up early

get是系动词，将get替换成be动词，形成I am up early 我起床很早，这里 up是形容词，意思为“起床的”

例子：He stands up to grap the ping-pong ball

句中的up是副词，修饰stands，意思为“向上”。

相似的用法：The police looked in./A bird flew over.

而into只有介词词性，没有副词词性，所以下面的句子是错误的：The police looked into.

例子：The guy lifted the wooden box up.

这句话中up为补语，类似的用法：Can you turn the light <u>off</u>?/ You may keep him <u>out</u> the whole night.



#### 部分倒装语序 vs 介词短语修饰动词

在S+V+O+C句型中，如果宾语的长度比补语还要长，则有时可以将宾语后置，形成部分倒装语序。

The guy is lifting the wooden box up. = The guy is lifting up the wooden box.

Can you turn the light off? = Can you turn off the light?

注意上面的句子都不是介词短语修饰动词

如何判断？将介词还原到补语位置，在宾语和补语中加入be动词

Tom turned on the radio = Tom turned the radio on = The radio was on  （部分倒装语序）

Tom jumped on the bus = Tom jumped the bus on = The bus was on （介词短语修饰动词）



### 分词

现在分词

现在分词本质上不代表现在的时间，而是一种进行态。

现在分词和动名词形式一致但成分属性不一致



## 从句

### 名词性从句

名词性从句-->名词-->主语、宾语、表语、补语、介宾

疑问句的改写：

1. 将句子中需要提问的部分替换成相应的疑问代词或疑问副词，并将其置于句首
2. 将助动词放在疑问代词或疑问副词后面，并将处于句尾的句号改为问号

名词性从句的改写：

1. 参照上面疑问句的改写，但不用将助动词提前
2. 名词性从句不能单独成句，疑问句可以
3. 除了疑问代词和疑问副词，还有三个词可以作为名词性从句的引导词：that\whether\if；使用这三个词时，直接将其置于简单句的句首就行
4. whether 和 if的词义均为“是否”，一般情况下两者可以互换使用，但whether可以引导作主语的名词性从句，if不可以
5. that作为引导词没有任何语义，
   1. 其引导的名词性从句在S+O+V句型的宾语位置时，that可以省略：
      1. I knew ~~that~~ Tom opened the box with a knife.
   2. 在S+V+P句型中P为形容词时，其后名词性从句中的that也可以省略：
      1. I'm afraid ~~that~~ you have been late.（介宾省略of，因为不能将that引导的名词性从句直接置于介词之后）

#### it的用法

1. 用作代词：
   - The book is mine. It is very interesting.
2. 用作形式主语，将较长的结构后置（名词性从句，动名词和动词不定式）：
   - That we shall succeed is certain. = It is certain that we shall succeed.
3. 用作形式宾语，将较长的结构后置（名词性从句，动名词和动词不定式）：
   - Shelly made(明确) that she would not go clear. = Shelly made it clear that she would not go.
4. Dummy It 只有象征意义，It不指代任何特定的对象，本身也没有任何词义，它唯一的作用就是在句中填补主语或宾语的位置：
   - It is dark outside.
   - It seems that he has been there for years.
   - Can you make it to the party?
5. 用于构成强调句：It + be动词 + 强调部分 + that
   - It was Tom that opened the box with the knife.

### 定语从句

定语从句 -> 形容词 -> 定语、表语、补语

定语从句的改写：

1. 将句中需要改写的部分（主语）替换为相应的关系代词或关系副词，然后将这个关系代词或关系副词置于句首

引导词：

| 引导词   | 替代对象             |
| -------- | -------------------- |
| who/whom | people               |
| which    | objects              |
| that     | people or objects    |
| where    | place                |
| when     | time                 |
| whose    | possessive（所有格） |

1. whom 和 who用法相同，但whom更常用于书面语，且专门用于替换宾语（介宾）位置上的people。在非正式常用中一般用who替换people.
2. who替换的对象的people，which替换的对象是objects，而that则可以替换people或objects
3. 

who\whom\that\which 用于替换句子中的宾语和介宾（只能是人或者五）时，则可以将这个引导词省略

例：Those are apples ~~which/that~~ I bought yesterday.

# 句子成分

## 主干成分

### 主语（S)

### 谓语（V）

### 宾语（O）

#### 宾补（Op）

### 表语（P）

简单结构：名词（+代词）、形容词

- 

复杂结构：介词短语、动词不定式、分词、定语从句、名词性从句



跟在be动词之后做表语

- 上面所有的所有名词性结构和形容词性结构
- 部分副词（可以被看做介词短语的变形）
  - here、there、elsewhere、everywhere、somewhere
  - everywhere = in every where
- 当省略be动词后，原句中的表语变为了修饰主语的定语





### 补语（C）

补语：有一类动词会改变宾语的动作或状态，对发生变化后的动作或状态进行补充说明的成分称为补语

补语的特点1：补语是句子中不可省略的成分

补语的特点2：补语能够与宾语组成一个完整的句子，且该句阐述的信息与原句表达的信息基本相同

其他1：在中文中，宾语和补语可以直接构成句子，而在英语中，两者中间还需要加上系动词；

其他2：补语可以转换成“宾语+be动词+补语（原来的补语，现在为表语）”的，换句话说，补语是名词性从句被省略be动词得来的

其他3：从上可知，补语是表语演变来的，故补语也只能由名词和形容词充当

## 修饰成分

### 定语

简单结构：形容词 

- 置于被修饰词之前或之后

复杂结构：介词短语、动词不定式、分词、定语从句

- 只能用于修饰名词或代词
- 需紧跟在被修饰词之后

### 状语

简单结构：副词

- 置于句首、句中（被修饰词之**前**）、句末

复杂结构：介词短语、状语从句、动词不定式、分词

- 置于句首、句中（被修饰词之**后**）、句末

- 一般只用于修饰动词、形容词和句子
- 状语从句：
  - 只能修饰句子
