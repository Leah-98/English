# 一、四大词性

动词、名词、形容词、副词

## 【动词】

任何复杂结构都不能作为动词使用

### 分类

#### 系动词▲

**系动词：**本身不表示任何动作，用于将前后两个结构联系起来，主语+系动词之后的内容为表语。

**常用的系动词：**be动词、keep、seem、appear、become、feel、get、grow、remain、smell、look、sound、taste、stay、go

**be动词：**is、are、am、was、were、been、be、being

**特点：**

1. 之后可以接名词和形容词（表语只能为名词或形容词）

2. 将系动词替换为be动词，句子的语义基本不变

#### 助动词

不能用作谓语动词，不具有实际意义

#### 情态动词

不能用作谓语动词，不具有实际意义

### 使用

#### 时态▲

|      | 一般式             | 进行式                     | 完成式                   | 完成进行时                          |
| ---- | ------------------ | -------------------------- | ------------------------ | ----------------------------------- |
| 现在 | 一般现在时 do      | 现在进行时 am/is/are doing | 现在完成时 have/has done | 现在完成进行时 have has been doing  |
| 过去 | 一般过去时 did     | 过去进行时 was/were doing  | 过去完成时 had done      | 过去完成进行时 had been doing       |
| 将来 | 一般将来时 will do | 将来进行时 will be doing   | 将来完成时 will had done | 将来完成进行时 will have been doing |

注意：

1. 一般现在时表述客观事实、习惯等，现在进行时表述进行态

2. 现在完成时重视动作的结果，一般过去时重视动作发生的时间

3. 如果要明确表示两个过去动作发生的先后顺序，使用过去完成时表述先发生的动作（过去的过去），使用一般过去时表述后发生的动作（过去）：
   - When I arrived home, Tom had left



##### 虚拟语气▲

用法：

1. 将句子中的谓语后退一个时态；过去完成时 had done <-- 一般过去时 did <-- 一般现在时 do

2. 无论主语是单数还是复数，在虚拟语气中只能将谓语be动改写为was

3. 在谓语动词前加上情态动词 would/might/should/could 表示对”将来发生动作“的虚拟语气

4. if...(then)结构的虚拟语气：

   - 【现在】if + 一般过去时，then + would/might/should/could + 动词原形

   - 【完成】if + 过去完成时，then + would/might/should/could + 现在完成时

   - 【将来】if + should/were to + 动词原形，then + would/might/should/could + 动词原形


5. 主句和从句也可以表达对不同时间的虚拟：
   - If they had set off yesterday(过去), they would be here now(现在). 
   - If we shouldn't have a date this afternoon(将来), I would go shopping now(现在).



#### 主谓一致▲

主语为**不可数名词**时，谓语动词只能为单数



**集体名词**是用于表示一类人或者一类事物，比如familiy\team\company\crowd\class\couple

- 当这个集体被作为一个整体看待时，谓语动作应为单数
  - 例如：My family is going to travel abroad.
- 当集体中的成员被作为个体看待时，谓语动词应为复数
  - 例如：My family are taking different planes to Paris.



由于**动名词、动词不定式和名词性从句**这三类名词性结构均没有复数形式，所以当这三类结构用作主语是，谓语动词只能为单数形式



**不定代词**：

- one\either\neither\each 表示单个的人或事物，谓语动词为单数
- every-\no-\some-\any- 复合不定代词，表示单个的人或事物，谓语动词为单数
- many\both\few\others 表示多个人或事物，谓语动词为复数
- some\any\none\all\more\most 表示单数又能表示复数，谓语动词的形式由介词of后的名词或代词决定
  - All of my friends like the movie. All of the money was wasted on the project.
  - 在"X of Y"的结构中，并不会将其中的 of Y 看作介词短语，而会将 X of 看作一个修饰Y的形容词



**主语连接：**

- 当主语是and连接的两个（或以上）的事物时，谓语动词使用复数形式
- 当主语是or或者nor连接的两个（或以上）的<u>单数事物</u>时，谓语动词使用单数形式
- 当主语是在单数事物或者复数事物之间选择时，谓语动词与更靠近它的那个事物单复数形式一致



#### 被动语态▲

| 时态       | 主动语态        | 被动语态             |
| ---------- | --------------- | -------------------- |
| 一般现在时 | do              | is/am/are done       |
| 一般过去时 | did             | was/were done        |
| 一般将来时 | will do         | will be done         |
| 现在进行时 | is/am/are doing | is/am/are being done |
| 过去进行时 | was/were doing  | was/were being done  |
| 现在完成时 | have/has done   | have/has been done   |
| 过去完成时 | had done        | had been done        |

被动语态中be动词的时态与主动语态中谓语动词的时态应保持一致，而被动语态中过去分词的形态永远是done



##### used的用法▲

1. 形容词，“习惯的”：
   - I am used to the hot days.
2. 动词，“过去常做的”
   - He used to go shopping during the weekend.
3. 动词，“使用”
   - The internet can be used to do what traditional advertising does.



## 【副词】

**修饰对象：**可以修饰名词和代词之外的其他结构。

**放置位置：**

当修饰形容词、副词、短语、动词不定式等时，一般置于句中（被修饰词之前）

当修饰句子时，一般置于句首

当修饰动词时
- 程度副词、频率副词：置于句中（谓语之前）
  - 例外：句中的谓语是be动词时，频率副词只能置于谓语之后
- 方式副词、时间副词、地点副词：置于句末
  - 例外：为了让别人知道方式副词在句中修饰的到底是什么结构，有时也会让副词尽量靠近其修饰的结构：
    - The boy placed cherries on the cake carefully --> The boy carefully placed cherries on the cake

### 	分类

- 评论副词：表示说话者的态度和意见
  - 必须放在句首
  - 必须用逗号将其与句子分开
  - 比如：fortunately\finally
- 方式副词：为动词添加“该动作发生的方式”这个附加信息
  - 即回答how开头的问题
  - 一般用于修饰动词
  - 比如：fast\quietly
- 频率副词：为动词添加“该动作发生的频率”这个附加信息
  - 即回答how often开头的问题
  - 一般用于修饰动词
  - 比如：sometimes\never
- 程度副词：为目标词添加“程度”这个附加信息
  - 即回答how much开头的问题
  - 一般用于修饰形容词、副词
  - 比如：very\quite\rather(相当)
- 时间副词：为动词或句子提供“该动作发生的时间”这个附加信息
  - 即回答when开头的问题
  - 一般用于修饰动词、句子
  - 比如：today\tomorrow
- 地点副词：为动词或句子提供“该动作发生的地点”这个附加信息
  - 即回答where开头的问题
  - 一般用于修饰动词、句子
  - 比如：outside\here
- 连接副词(特殊)：
  - 并不修饰任何结构，作用是连接前后两个句子
  - 不可以删除，会影响语义
  - 通常用逗号将其与后面的句子分离
  - 常用的包括：however\therefore\nevertheless\moreover\thus



### Still Yet Already▲

三者是时间副词：

- Still：一件事在”现在“还在进行

- Yet：一件事在”将来“某个时间才会发生
  - 注意：通常只用于否定句或疑问句

- Already：一件事在”过去“已经发生了



# 二、句子成分

**主干成分：**主语（S)、谓语（V）、宾语（O）、介宾（Op）、表语（P）、补语（C）

**修饰成分**：定语、状语

**四大词性与七类成分的关系：**

- 名词：主语(S)、宾语(O)、表语(P)、补语(C)
- 形容词：定语、表语(P)、补语(C)
- 副词：状语
- 动词：谓语(V)

## 【表语】

表语 -> 名词(+代词) 、形容词  ->介词短语、动词不定式、分词、定语从句、名词性从句



可以跟在be动词之后做表语的：

- 上面所有的所有名词性结构和形容词性结构
- 部分副词（可以被看做介词短语的变形）
  - here、there、elsewhere、everywhere、somewhere
  - everywhere = in every where

特点：

1. 及物动词之后的结构为介宾(Op)、系动词之后的结构为表语

2. 当省略be动词后，原句中的表语变为了修饰主语的定语



## 【补语】

有一类动词会改变宾语的动作或状态，对发生变化后的动作或状态进行补充说明的成分称为补语



补语 -> 名词(+代词) 、形容词  ->介词短语、动词不定式、分词、定语从句、名词性从句



特点：

1. 补语是句子中不可省略的成分

2. 补语能够与宾语组成一个完整的句子，且该句阐述的信息与原句表达的信息基本相同

3. 在中文中，宾语和补语可以直接构成句子，而在英语中，两者中间还需要加上系动词
4. 补语可以转换成“宾语+be动词+补语（原来的补语，现在为表语）”的，换句话说，补语是名词性从句被省略be动词得来的

5. 从上可知，补语是表语演变来的，故补语也只能由名词和形容词充当



## 【定语】

定语 -> 形容词 -> 介词短语、动词不定式、分词、定语从句

**放置位置：**

- 简单结构：形容词 
  - 置于被修饰词之前或之后


- 复杂结构：介词短语、动词不定式、分词、定语从句

  - 只能用于修饰名词或代词

  - 需紧跟在被修饰词之后




## 【状语】

状语 -> 副词 -> 介词短语、状语从句、动词不定式、分词

**放置位置：**

- 简单结构：副词
  - 置于句首、句中（被修饰词之**前**）、句末


- 复杂结构：介词短语、状语从句、动词不定式、分词

  - 置于句首、句中（被修饰词之**后**）、句末


  - 一般只用于修饰动词、形容词和句子

  - 状语从句只能修饰句子




# 三、七类复杂结构

七类复杂结构：

- 介词短语：形容词、副词

- 从句
  - 名词性从句
  - 定语从句
  - 状语从句
- 非谓语动词
  - 动名词：名词
  - 动词不定式（to+V）：名词、形容词、副词
  - 分词（V+ing 现在分词、V+ed 过去分词）：形容词、副词



七类复杂结构与四大词性的关系：

- 名词：动词不定式、动名词、名词性从句
- 形容词：介词短语、动词不定式、分词、定语从句
- 副词：介词短语、动词不定式、分词
- 动词



## 【介词短语】

介词是用来建立一个名词与其他词（包括名词、动词、形容词等）关系的一类词

介词短语：介词+名词（名词性结构），组成表示时间、状态、目的、原因的介词短语



### at\on\in▲

**表示地点时的区别**

at：一维、表示点

on：二维、表示平面的表面

in：三维、表示物体的内部

当乘坐飞机、火车、公共汽车等大型交通工具时，也可以看作是位于一个较大的平台表面之上，可以使用on：on the plane\on the train\on the bus

当使用介词in时，通常特指在某个事物或区域的内部：in the library

当表示乘坐私家车、小船等小型交通工具时，我们会突出人所处的位置是在这类交通工具的内部，因此也会使用介词in：in the car\ in the boat



**表示时间时的区别**

这三个介词按照 at ->on->in 的顺序，表示的时间长度逐渐增加。

介词at通常用于表示一天中某个特定的时间，即“某个时间点”：at 8:00\ at the moment

介词on通常用于表示“某一天”：on Tuesday\on Valentine's Day

介词in通常用于表示一个更长的时间段，比如周、月、年、世纪等：in 1998\in September



### 成分判断及补充说明▲

**判断介词短语在句子中的成分**

假设其时定语、状语、补语中的任意一类，翻译成中文后看看是否符合逻辑

例子：Dad put a parcel under the table

1. under the table是定语：修饰 a parcel，a parcel under the table翻译后为”桌子下面的包裹“，整句为”爸爸放桌子下面的包裹“
2. under the table是状语：状语可以省略，省略后为Dad put a parcel，有语义问题
3. under the table是补语：可以省略主语和谓语，在宾语和补语之间加入be动词后得到a parcel was under the table，翻译整句为”爸爸把包裹放在了桌子下“

根据上面的分析，under the table应该为补语



**补语对介宾做补充说明**

称介词短语中的名词为介宾，因为介宾是宾语中的一种，补语同样可以对介宾作补充说明。

例子：I looked at Andy sitting there helplessly

上面的例子中，sitting there helplessly为 Andy的介宾补语，可以与Andy组成句子：Andy was sitting there helplessly



**可以作为介词的词，其实使用了其他词性的情况**

例子：I get up early

get是系动词，将get替换成be动词，形成I am up early 我起床很早，这里 up是形容词，意思为“起床的”

例子：He stands up to grap the ping-pong ball

句中的up是副词，修饰stands，意思为“向上”。

相似的用法：The police looked in./A bird flew over.

而into只有介词词性，没有副词词性，所以下面的句子是错误的：The police looked into.

例子：The guy lifted the wooden box up.

这句话中up为补语，类似的用法：Can you turn the light <u>off</u>?/ You may keep him <u>out</u> the whole night.



**部分倒装语序 vs 介词短语修饰动词**

在S+V+O+C句型中，如果宾语的长度比补语还要长，则有时可以将宾语后置，形成部分倒装语序。

The guy is lifting the wooden box up. = The guy is lifting up the wooden box.

Can you turn the light off? = Can you turn off the light?

注意上面的句子都不是介词短语修饰动词

如何判断？将介词还原到补语位置，在宾语和补语中加入be动词

Tom turned on the radio = Tom turned the radio on = The radio was on  （部分倒装语序）

Tom jumped on the bus = Tom jumped the bus on = The bus was on （介词短语修饰动词）



## 从句

### 【名词性从句】

**名词性从句-->名词-->主语、宾语、表语、补语、介宾**



**疑问句的改写：**

1. 将句子中需要提问的部分替换成相应的疑问代词或疑问副词，并将其置于句首
2. 将助动词放在疑问代词或疑问副词后面，并将处于句尾的句号改为问号

**名词性从句的改写：**

1. 参照上面疑问句的改写，但不用将助动词提前
2. 名词性从句不能单独成句，疑问句可以
3. 除了疑问代词和疑问副词，还有三个词可以作为名词性从句的引导词：that\whether\if；使用这三个词时，直接将其置于简单句的句首就行
4. whether 和 if的词义均为“是否”，一般情况下两者可以互换使用，但whether可以引导作主语的名词性从句，if不可以
5. that作为引导词没有任何语义，
   1. 其引导的名词性从句在S+O+V句型的宾语位置时，that可以省略：
      1. I knew ~~that~~ Tom opened the box with a knife.
   2. 在S+V+P句型中P为形容词时，其后名词性从句中的that也可以省略：
      1. I'm afraid ~~that~~ you have been late.（介宾省略of，因为不能将that引导的名词性从句直接置于介词之后）



#### it的用法▲

1. 用作代词：
   - The book is mine. It is very interesting.
2. 用作形式主语，将较长的结构后置（名词性从句，动名词和动词不定式）：
   - That we shall succeed is certain. = It is certain that we shall succeed.
3. 用作形式宾语，将较长的结构后置（名词性从句，动名词和动词不定式）：
   - Shelly made(明确) that she would not go clear. = Shelly made it clear that she would not go.
4. Dummy It 只有象征意义，It不指代任何特定的对象，本身也没有任何词义，它唯一的作用就是在句中填补主语或宾语的位置：
   - It is dark outside.
   - It seems that he has been there for years.
   - Can you make it to the party?
5. 用于构成强调句：It + be动词 + 强调部分 + that
   - It was Tom that opened the box with the knife.



### 【定语从句】

**定语从句 -> 形容词 -> 定语、表语、补语**



**定语从句的改写：**

1. 将句中需要改写的部分（主语）替换为相应的关系代词或关系副词，然后将这个关系代词或关系副词置于句首



**引导词：**

| 引导词   | 替代对象             |
| -------- | -------------------- |
| who/whom | people               |
| which    | objects              |
| that     | people or objects    |
| where    | place                |
| when     | time                 |
| whose    | possessive（所有格） |

注意：

1. whom 和 who用法相同，但whom更常用于书面语，且专门用于替换宾语（介宾）位置上的people。在非正式常用中一般用who替换people.
2. who替换的对象的people，which替换的对象是objects，而that则可以替换people或objects
3. which/that we can dance in = where we can dance
4. that还可以跟在形容词之后表示这个名词的具体内容
   - We reached the decision that Tom would remain the captain. -> The decision was that Tom would remain the captain.



#### 限制性定语从句和非限制性定语从句▲

1. 限制性定语从句：缩小了名词的表示范围
2. 非限制性定语从句：名词的指向已经非常明确或表示的对象已经是唯一了。此类从句并不直接修饰名词或代词，只是为名词或代词提供一些附加信息

**引导词which和that的选择：**

1. 理论上说which/that都是正确的，因为两者在引导限制性定语从句时都是正确的
2. 但在日常应用中，我们会用that来引导限制性定语从句，并将它看作限制性定语从句的一个典型标志
3. 也有特例，如果有介词前置的情形，必须使用which
   1. Can you help to find a room in which we can dance. (这里的which不可以替换成that)
4. 无论在什么情况下，that都不能用作非限制性定语从句的引导词
   1. Apples, which are good for health, should be eaten every day. (这里的which不可以替换成that)
5. 通常使用that引导限制性定语从句，使用which引导非限制性定语从句



#### **定语从句的省略▲**

1. who\whom\that\which 用于替换句子中的宾语和介宾（只能是人或者物）时，则可以将这个引导词省略
   - Those are apples ~~which/that~~ I bought yesterday.
2. 只要定语从句是S+V+P句型，且引导词为which/who/that时，我们都可以通过省略引导词和be动词来达到简化从句的目的
   - Do you know the man ~~that is~~ in the car.
3. 如果引导词后并不是be动词，也可以将定语从句的时态改写成进行时，创造出一个be动词然后进行省略
   - that costs $1500 a month -> that is costing $1500 a month -> The company has made an insurance plan costing $1500 a month



### 【状语从句】

**状语从句 -> 副词 -> 状语**



**状语从句的结构：**从属连词+简单句



**并列连词和从属连词：**

并列连词：表示多个单词、短语、句子的并列关系；for\and\nor\but\or\yet\so(FANBOYS)

从属连词：用于构成状语从句，从属连词+简单句=状语从句；thoughif\\\in case\as soon as\as if\so that

注意：不要用逗号代替连词



**从属连词决定状语从句的属性**

1. 时间属性：when\while\before\after
2. 条件属性：if\as long as\even if\
3. 原因属性：because\as\since
4. 对比属性：although\even though\whereas\while



#### 状语从句的省略▲

1. 如果状语从句的主语和主句的主语相同，则省略状语从句中的主语和它后面的be动词
   - While ~~I was~~ reading this book, I picked up more than one hundred printing mistake.

2. 当从属连词为while和when时，在不影响语义的前提下，也可以将从属连词一起省略
   - ~~While~~ reading this book, I picked up more than one hundred printing mistake.

3. 表示"原因"的状语从句必须将从属连词一起省略

4. 如果状语从句的主语和主句的主语不同，需要构成独立主格结构后进行省略，不然会形成垂悬修饰语

   - <u>Staring at him,</u> he could not do anything.(垂悬修饰语)

   - <u>The girl staring at him</u>, he could not do anything.(独立主格结构)

## 非谓语动词

非谓语动词都可以被看作省略了主语的从句

### 【分词】

**分词 -> 形容词、副词 ->表语、定语、补语、状语**



#### 分类

##### **现在分词**

1. 现在分词本质上不代表现在的时间，而是一种进行态。

2. "-ing"为后缀的分词描述给人这种感受的事物

3. 现在分词和动名词形式一致但成分属性不一致

##### **过去分词**

1. 过去分词本质上不代表过去的时间，而是一种完成式和被动语态。

2. "-ed"的分词描述一个人的感受 



#### 成分

##### 用作表语

现在分词+be动词搭配成进行态；过去分词+be动词搭配成被动语态



##### **用作定语▲**

分词作为单个词使用时，需要将其置于名词之前

分词不仅仅包含单个的词时，需要将其置于名词或代词之后：

- The girl <u>standing over there</u> is very charming.

为什么分词会不仅仅包含单个的词？因为这类分词是定语从句省略后得来的：

- His sister, who is studying in Peking University, is a volunteer. = His sister, studying in Peking University, is a volunteer.

非限制性定语从句中的which不仅仅能指代事物，而且能指代整个主句：

- Tom pretended not to know me, which was making me embarrassed. = Tom pretended not to know me, making me embarrassed.

这就是说，我们可以将一个现在分词放在主句之后，用于表示主语造成的影响或结果：

- Trains were held up by a heavy snow, causing many people late for work.



##### 用作补语▲

现在分词一般在watch/see/feel/hear等感官动词之后用作补语

过去分词一般在keep/have/get等动词之后用作补语



##### 用作状语▲

分词用作状语时修饰的对象是形容词、动词和句子，作用是为被修饰结构提供时间、原因和条件等附加信息

可以用分词明确表示两个动词的先后顺序，将分词中动词的时态改成完成时，用完成时表示这个动作在主句之前已经发生了：

- Having found the cause, the doctors were able to treat the disease.

分词用作状语时，它的逻辑主语和主句的主句必须相同，不同的话，此时的分词就是**垂悬修饰语**了，比如：

- Opening the window, stars were calling my name.(错误)

- Opening the window, I heard stars calling my name.(正确)

动词不定式也可能变成垂悬修饰语：

- To play tennis well, a good racket is needed.

- To play tennis well, I need a good racket.

放置位置：

1. 修饰句子：放置在句首，用逗号隔开

2. 修饰形容词：句中（被修饰词之后）

3. 修饰动词：通常表示时间，即表示一个与修饰动词同时发生的动作，应位于句中（被修饰词之后）或句末



### 【动名词】

本质是省略了主语的名词性从句 

- Going to bed early is a good habit of mine. = That I go to bed early is a good habit of mine.

**动名词->名词 -> 主语、宾语、表语、补语、介宾**

#### **通过改变动名词的逻辑主语**▲

- Being Late will make the teacher rather upset. = My being late will make the teacher rather upset.

- I am worried about staying up so late every day.

- I am worried about my daughter's staying up so late every day.



### 【动词不定式】

本质是省略了主语的从句



#### 动词不定式改写为从句时的特别注意事项▲

1. 需要补上动词不定式的逻辑主语

   - John needs a pen to write with. = John needs a pen that he can write with.

   - He asked his daughter to assist him. = He asked his daughter. + His daughter was to assist him. 
   - To know you is my pleasure. = That I could know you is my pleasure.
   - To carry out the project immediately is impossible. = (不确定主语，使用不定代词ones/one作为逻辑主语)That ones are to carry out the project immediately is impossible.

1. 动词不定式通常表示发生在将来的动词，所以改写时需要在谓语前加入will(be to)\would\can\could\should
   - I am glad to meet you. = I am glad that I could meet you.



动词不定式 -> 名词、形容词、副词 -> 除了谓语、介宾外的任何一类成分

#### 成分

##### 用作表语

1. 表示主语的具体内容：
   - His solution was to create another workshop.
2. 表示计划做的事情：
   - Some of them were to learn dancing.

##### 用作补语▲

动词不定式和动名词都可以跟在及物动词后做宾语，那么什么时候该用to do 什么时候该用doing?

1. forget/remember/regret： to do表示需要去做的事，doing表示已经做过的事

2. like/love/hate/prefer/start/continue：使用 to do / doing 没有区别

3. enjoy/avoid/dislike/finish：只能接doing

4. plan/want/decide/hope/would like：只能接to do



**利用 for + 名词/代词 结构可以改变动词不定式的逻辑主语**

- I consider it necessary for you to go with him.



how to do\ what to do 本质上是名词性从句省略后得来的：

- what he should do = what to do 



##### 用作定语▲

需置于名词或代词之后



如果定语从句的引导词that替换的是主语或宾语，也可以将这个定语从句改写为动词不定式

1. 当that替换主语时：
   - My mother is the first one that will get up. = My mother is the first one to get up.

2. 当that替换宾语时：
   - She has a great many things that she should do. = She has a great many things to do.

3. 当that替换介宾时：
   - I have someone that I can think about. = I have someone to think about.



特殊用法：直接跟在名词用作定语，表示这个名词的具体内容。



##### 用作状语▲

状语从句只能修饰句子，动词不定式能够修饰动词、形容词和句子



动词不定式用作状语时，通常表示原因和目的，当它用作状语修饰句子、形容词和动词时，其逻辑主语和主句需要一样。



动词不定式用作状语时在句子的位置：

1. 修饰句子时应位于句首
2. 修饰形容词时应位于句中（被修饰词之后）
3. 修饰动词时应位于句中（被修饰词之后）或句末



动词不定式修饰动词引发的相似句型解析：

1. Everybody is excited to start the new season. 动词不定式为状语
2. The new manager is easy to get along with. = It is easy to get along with the new manager. 动词不定式为主语



# 【并列句】

并列句：利用并列连词将两个及以上简单句连接后构成的句子



如果并列句中呈现的是句子的并列时，需要在并列连词前加逗号

如果并列句中呈现的是（省略后的结构）句子的一部分、短语、单词的并列时，不需要在并列连词前加逗号



三个及以上子句的并列：A,B, and C / A, B, or C



**并列举的省略▲：**

- 将靠后子句的重复部分省略
  - We desire happiness, and we desire health = We desire happiness and health
- 当并列结构是从句时，应该保留从句的引导词
  - He told me that his parents had passed away and that he had to make a living by himself.
- 日常交流和沟通时的问答也可以看作多个简单句的并列，因此也可以将重复的内容省略
  - When is the next train for Beijing? ~~The next train for Beijing is~~ in half an hour.




并列连词的用法：

- either...or...：二选一
- neither...nor...：同时对两者的否定
- not only...but also...：(情绪上的)递进
- both...and...：同时包含两者



## 【比较句】

一种特殊的并列句



### as...as...

用于表示两个对象的某项特性相同或不同

第一个as为副词，意为”同样地“

第二个as为连词，意”像……一样“

如果上下文中已经明确指出了作比较的对象，则第二个as不必出现：

Tom does his homework till 10:00 pm every day. Tom studies as hard.

第二个as后的子句重复内容可以被省略，as even, as possible等短语也是并列句省略后的产物，但是省略不应产生歧义



### than 

一般用于表示两个对象某种特性的大小或多少

在词句型中，than为连词

如果上下文中已经明确指出了作比较的对象，则than不必出现：

John stands 1.8 meters tall. Tom is a head taller than John.

than后的子句重复内容可以被省略，但是省略不应产生歧义：

I like milk more than cats.(错误)



#### 比较级和最高级▲

比较级：表示一个对象和另一个对象相比较的结果

构成比较级的方法：

- 单音节词：通常在该词后加-er
- 多音节词：通常在该词前加more

最高级：表示一个对象与其他所有对象相比较的结果

构成最高级的方法：

- 单音节词：通常在该词后加-est
- 多音节词：通常在该词前加most
